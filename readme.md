# 🚀 To-do List

O projeto se trata de uma aplicação de uma to-do list onde é possível adicionar itens e suas descrições e manipulá-los. A aplicação também exibe o percentual de atividades marcadas como concluídas. São disponibilizados temas para que o usuário personalize a aparência do seu ambiente.

## 💻 Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de explorar as atividades de um CRUD. Através dele, busquei aplicar os seguintes conceitos a fim de elaborar uma solução eficiente e intuitiva para o usuário:

- HTML semântico
- HTML seo
- CSS responsivo para dispositivos móveis, tables e desktop
- Javascript: manipulação do DOM
- Javascript: módulos, importar e exportar funcionalidades
- Javascript: eventos

### ✨ Principais Funcionalidades:
- **Adicionar item na lista:** Adiciona um item com seu nome e sua descrição.
- **Editar item da lista:** Permite modificar um item existente na lista.
- **Excluir item da lista:** Permite excluir um item da lista.
- **Percentual:** Exibe o percentual de atividades marcadas como concluídas.
- **Temas:** Disponibiliza temas para personalizar a interface do usuário.

## 🚀 Como Executar o Projeto

Caso queira rodar este projeto localmente na sua máquina, siga os passos abaixo:

```bash
# 1. Clone este repositório
$ git clone https://github.com/marinaspdev/to-do-list.git

# 2. Acesse a pasta do projeto
$ cd to-do-list

# 3. Instale as dependências (se houver)
$ npm install

# 4. Execute a aplicação
# abra o index.html no navegador
```
